# Password Generator

A simple Password Generator built using HTML, CSS, and JavaScript.

## Features

- Generate secure random passwords
- Choose password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include symbols
- Copy password to clipboard

## Technologies Used

- HTML
- CSS
- JavaScript
